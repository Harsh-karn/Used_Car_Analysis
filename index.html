🚗 Used Cars Analysis

Comprehensive Data Analysis Dashboard & API

📊 Project Overview

This project provides a comprehensive analysis of used car data with interactive visualizations, machine learning insights, and a RESTful API. Built with Python, Pandas, and modern web technologies.

📈 Data Analysis

Comprehensive analysis of 6,019 used car records from 1998-2019, including price trends, brand comparisons, and market insights.

🤖 Machine Learning

Advanced correlation analysis and feature importance ranking to understand what drives car prices in the used car market.

📊 Interactive Visualizations

Dynamic charts and graphs powered by Plotly, showing price distributions, brand comparisons, and market trends.

🔍 Data Explorer

Interactive filtering and search capabilities to explore the dataset with custom filters for brands, fuel types, and price ranges.

🔌 API Endpoints

Access the data programmatically through our RESTful API:

GET /api/stats - Get dataset statistics
GET /api/brands - Get brand analysis
GET /api/years - Get year analysis
GET /api/fuel - Get fuel type analysis
GET /api/correlation - Get feature correlations
POST /api/search - Search cars with filters

🛠️ Technologies Used

  • Python - Core programming language
  • Pandas - Data manipulation and analysis
  • NumPy - Numerical computing
  • Scikit-learn - Machine learning algorithms
  • Plotly - Interactive visualizations
  • Flask - Web API framework
  • Streamlit - Interactive dashboard

📋 Dataset Information

  • Total Records: 6,019 used cars
  • Time Period: 1998-2019
  • Features: Price, Brand, Year, Fuel Type, Transmission, Owner Type, Engine Size, Power, Mileage, Location
  • Price Range: ₹0.44L - ₹160.0L
  • Unique Brands: 32 different car brands
